# Abra Chan - Das Hühnerhaus Community Guidelines

**Repository:** Public transparency for my streaming community.

This repo holds the official rules for 🐔 **Das Hühnerhaus** 🏡 — my Twitch, Kick, YouTube, and Discord community.

### Why Public + Git?
- Full transparency
- Version history
- Community feedback welcome
- Accountability

### Current Version
- **Version:** 1.1  
- **Last updated:** June 2026

### Core Philosophy
We want a **weird, positive, high-signal** space where:
- Honest talk and pattern recognition are allowed
- Constant identity grandstanding and virtue signaling are not
- People can vibe, meme, and talk about games/VTubing without turning every stream into a battlefield
- Cool people from all backgrounds are welcome as long as they don't make everything about their identity

### How to Contribute
1. Open an Issue with your suggestion
2. (Optional) Submit a Pull Request with changes
3. Discuss in the `#community-feedback` channel on Discord

I (Abra) have final say on changes, but I actually read and consider all feedback.
